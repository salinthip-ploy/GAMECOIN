# 👻 GAMECOIN (Ghost vs Zombies)

เกม 2D อาร์เคดเอาชีวิตรอดและเก็บเหรียญ พัฒนาด้วยภาษา Python และไลบรารี Pygame ผู้เล่นจะได้รับบทเป็นผีน้อยที่ต้องคอยหลบหลีกฝูงซอมบี้ พร้อมกับเก็บเหรียญเพื่อทำคะแนนให้ได้มากที่สุด!

## ✨ ฟีเจอร์หลัก (Features)
* **ระบบควบคุมผู้เล่น (Player Mechanics):** ควบคุมการเคลื่อนที่ของตัวละครผีได้อย่างอิสระ
* **ระบบศัตรู (Enemy AI):** ซอมบี้จะปรากฏตัวจากทั้งด้านบนและด้านข้างของหน้าจอเพื่อเพิ่มความท้าทาย
* **ระบบคะแนน (Scoring System):** บันทึกคะแนนปัจจุบัน และมีระบบบันทึกคะแนนสูงสุด (High Score) ลงในไฟล์อัตโนมัติ
* **ระบบเสียงและภาพ (Audio & Visuals):** มีการอัปเดตอนิเมชันของตัวละคร พื้นหลังแบบเคลื่อนที่ได้ และเสียงเอฟเฟกต์เมื่อเก็บเหรียญ

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
* **Python 3.x**
* **Pygame** (สำหรับการเรนเดอร์กราฟิกและจัดการระบบเกม)
